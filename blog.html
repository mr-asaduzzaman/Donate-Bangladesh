What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects. Each element, attribute, and piece of text in the document becomes a node in this tree, which JavaScript can interact with to dynamically change the content, structure, or styles of the web page.

How do you select an element from the DOM?

Elements in the DOM can be selected using various methods. Commonly, JavaScript's document.getElementById() is used to select an element by its ID, while document.querySelector() allows selection using CSS-like selectors. Additionally, document.getElementsByClassName() and document.getElementsByTagName() select elements by their class name or tag name, respectively.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to handle events for all of its child elements, instead of adding individual listeners to each child. It is useful because it minimizes memory usage and improves performance, especially when working with dynamic elements. This is achieved by taking advantage of event bubbling, where events triggered by child elements propagate up the DOM tree.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, you can use methods like setAttribute() and getAttribute(). For instance, you can change an image's source or a link's href. To manipulate styles, you can modify the style property of an element, such as element.style.color = 'red';, which directly alters the inline styles of the selected element.
